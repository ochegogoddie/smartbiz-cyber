# SmartBiz Cyber Cafe — Portfolio & Price List

This is a static portfolio and price-list site for SmartBiz Cyber Cafe. It includes:

- A neon-inspired landing hero
- Services & categorized price list (Internet, Printing, Design, Support, Financial)
- A quick pricing calculator for common items
- Contact section

How to open:

1. Open the `smartbiz` folder in your browser or local server.
2. Open `index.html` in a browser.

Files:

- `index.html` — main markup
- `style.css` — styles
- `script.js` — interactions (typing, navbar hide, calculator)

Feel free to customize prices, contact details, images and add real images to the `images/` folder.
